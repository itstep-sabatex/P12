## RazorPage JS DZ
### В проекті RazorPagesDemo на стрінці NomenclatureJS виконати наступне:
1. Змінии алгоритм редагування елемента, замінивши отримання даних з рядка таблиці на отримання елемента за допмогою API (NomenclaturesController).
2. Додати Insert nomenclature. В таблицю фронтенд може додаватись нативно або оновленням таблиці.
3. При втдаленні виводити попередження в діалоговому вікні.
4. Реалізувати механізм виділення рядка. (Допоміжні функції та CSS див. site.css,Nomenclature.js)
5. Використовуючи механізм виділення рядка реалізувати розміщення кнопок (Insert,Edit,Delete,Refresh) над таблицею
6. Добитися того щоб таблиця займала весь вільнтй простір і містила прокрутку. (сторінка має займати тільки екран без прокрутки)
7. Реалізувати фокусування на редагуємий або втавлений рядок. 

** вікна редагування,вставлення, попередження діалогові
