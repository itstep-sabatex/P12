# Домашня робота з проекту Кафе

До існуючого проекту з даного репозиторія додадати наступний функціонал:
- Для кожного Entity зробити окреме вікно де буде відображено список елементів;
- Забезпечити в списку можливість додавання нових даних.
- Забезпечити в списку можливість видалення даних (опція список виділених).
- Забезпечити в списку можливість редагування даних.
